##  Known Limitations

Since you're using SQLite on Vercel:
- Database resets on each deployment
- Data from seeders loads fresh each time
- For persistent data, consider:
  - PlanetScale (free MySQL)
  - Supabase (free PostgreSQL)
  - Vercel Postgres
